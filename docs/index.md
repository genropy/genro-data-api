## Overview

genro-data-api exposes database data through industry-standard protocols:

- **OData v4** — query, metadata, filtering, pagination, relations
- **GraphQL** — planned for future releases

The package is server-agnostic: it works with both WSGI (GenroPy current)
and ASGI (genro-asgi future).
